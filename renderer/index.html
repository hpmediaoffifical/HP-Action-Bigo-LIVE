Cửa sổ overlay

Tên Màu nền Độ đặc Kích thước Vị trí Always-top Click-through Thao tác
Overlay là cửa sổ riêng phát mp4/webm/mp3. Đặt trong OBS bằng Window Capture (chọn cửa sổ Overlay) → thêm filter Color Key với chroma đúng màu nền. Mặc định 540×960. Bấm "Hiện" để mở cửa sổ rồi kéo/resize tự do — vị trí được lưu tự động. Khi sẵn sàng stream, bật 🔒 Click-through — overlay không cản chuột, không hiện taskbar, nhưng OBS vẫn capture và hiệu ứng vẫn play.
Hỗ trợ nhiều overlay chồng lên nhau (mỗi quà map vào 1 overlay riêng).

⚙️ Cài đặt chung

Mặc định TẮT cho tất cả quà. Vào dialog "Sửa quà" tick riêng từng quà cần phát pre-effect. Toggle Bật/Tắt ô này nhiều lần KHÔNG tự bật lại quà nào — mỗi quà giữ trạng thái user đã set.
Nhạc nền sẽ phát loop, có thể chọn output device (loa ngoài hoặc cable virtual cho OBS). Khi gift bật "Tạm dừng nhạc nền khi phát", nhạc nền tự pause trong lúc effect chạy rồi resume.

💾 Xuất / Nhập cấu hình

Xuất: tạo 1 file .json chứa toàn bộ danh sách quà, nhóm, overlay, cài đặt chung, BGM, pre-effect — KHÔNG bao gồm file media (mp3/mp4). Mang file này sang máy khác rồi Nhập là khôi phục mọi thứ. Đường dẫn file media là path local nên cần copy file media riêng nếu chuyển máy.
